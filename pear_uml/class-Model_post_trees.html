Class Model_post_trees

public class Model_post_trees

A Model class to represent the tree structure of posts of multiple topics.
This model uses the adjacency list model to represent trees.
Here each node(record) has a column dedicated to identifying the "parent" node.
Using this "parent" node it will become possible to walk trought the tree up to the root node.

Besides the columns from the table post_trees, the object in the array also contain "level" which reresents the nesting level.
