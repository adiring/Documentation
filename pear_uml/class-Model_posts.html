get_all_posts

public  get_all_posts(integer $post_id)

reads recursivly all responses to the declared "post_id" of the topic.
the resulting array contains all columns from the table posts (id, post_id, user_id, date, title, content).
Additional, the objects in the array also contain "level" which reresents the nesting level and "num_replies".
Note: the array also includes the first post (topic)

get_all_posts_as_tree

public null get_all_posts_as_tree(integer $post_id)

reads recursivly all responses to the declared "post_id" of the topic.
the resulting tree contains all columns from the table posts (id, post_id, user_id, date, title, content).
Additional, the objects in the tree also contain "level" which reresents the nesting level.
Note: the tree also includes the first post (topic) This function does essentially the same as get_all_posts(). However the result is returned as tree structure, instead of an array
