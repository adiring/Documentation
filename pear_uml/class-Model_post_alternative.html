get_tree_id_between

private  get_tree_id_between(string $lower, string $upper)

returns an id between lower and upper as string.
e.g. for '1' and '3' it would return '2'.
for '1' and '2' it would return '10'. If these ids are compared as strings '10' would be greater than '1' and smaller than '2'.
